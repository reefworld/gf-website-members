== Description ==

Get Green Fins Members from API, currently pulling data from the Green Fins Portal but in the future may pull from other sources.

= Auth: =
Send key as "key" post or get data

= Endpoints: =
* https://portal-dev.greenfins.net/api/v1/top-10-members
* https://portal-dev.greenfins.net/api/v1/countries/{country_id}/top-10-members
* https://portal-dev.greenfins.net/api/v1/countries
* https://portal-dev.greenfins.net/api/v1/countries/{country_id}/regions
* https://portal-dev.greenfins.net/api/v1/regions/{region_id}/locations
* https://portal-dev.greenfins.net/api/v1/locations/{location_id}/members
* https://portal-dev.greenfins.net/api/v1/members/{member_id}/last-assessment
